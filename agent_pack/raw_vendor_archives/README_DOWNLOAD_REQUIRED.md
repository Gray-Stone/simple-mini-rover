# Raw vendor archives

This folder is intentionally mostly empty in this generated pack.

The archive URLs were found, but the current tool environment blocked saving `.zip` and `.rar` MIME types directly. To populate this folder, run:

```bash
../scripts/download_vendor_archives.sh
```

Expected key archives:

- `WAVE_ROVER_demo.zip` - factory WAVE ROVER Arduino/ESP32 demo, expected to contain `WAVE_ROVER_v0.9.ino` and companion files.
- `WAVE_ROVER_DXF.rar` - chassis DXF drawing.
- `WAVE_ROVER_PDF.rar` - chassis PDF drawing.
- `WAVE_ROVER-EP_DXF.rar` - expansion/mounting plate DXF drawing.
- `WAVE_ROVER-EP_PDF.rar` - expansion/mounting plate PDF drawing.
- `WAVE_ROVER_MODEL_STL.rar` - STL model archive.
- General Driver board ZIP resources for board mechanical drawings and STEP model.
