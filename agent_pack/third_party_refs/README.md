# Third-party references

These are not vendor-authoritative sources, but may be useful for future agents:

- `winetree94/waveshare-rover-development-setup`: personal backup of scattered Waveshare rover Arduino setup files; repository lists `examples/WAVE_ROVER.zip`, required libraries, and ESP32 Arduino package 1.0.6.
- `antlassagne/ros2-wave-rover`: ROS 2 serial driver for WAVE ROVER-style JSON/UART control.
- `msanterre/wave_rover_serial`: Python package for Wave Rover serial commands.

Treat these as convenience references, not as hardware truth. Prefer Waveshare wiki, schematic PDF, product page, and actual board inspection for authoritative information.
